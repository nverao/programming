В Данной задаче был релизованан To-Do List, который является статическим массивом 
структур Tasks.

Cтруктура "Tasks" имеет поля:
name(непосредственно задача), priority(приоритет в списке), done(показывает, выполнена ли задача).

Были написаны функции:
1.createTask, которая создает задачу.
2.printTask, которая выводит задачу.
3.taskToString, которая переводит задачу из типа structure в тип структуры данных (но я не знаю насколкьо правильно
я оперирую этими словами ) string.
4.getTaskFromUser, которая получает задачу от пользователя(через терминал).
5.markAsDone, которая помечает задачу выполненой.
6.printAllTask, которая выводит весь массив задач.
7.setPriority, воспользовавшись которой пользователь задает приоритетность задаче.
8.saveToFile, которая сохраняет задачу в нужный файл.
9.swap(вспомогательная), которая меняет местами задачи в массиве.
10.printTasksInOrder, которая выводит задачи в соответствии с приоритетом.
11.importFromFile, которая импортирует задачи из файла.
12.deleteTask, которая удаляет задачу.

Чтобы вызывть нужную функцию одной цифрой, я использовала конструкцию switch-case.
В файле output.txt лежат несколько образных tasks(это имена котиков моей бабушки и брата),
чтобы можно было потестировать программу.
